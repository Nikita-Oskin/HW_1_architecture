# Hello World App

Простое веб-приложение, отображающее текст «Hello World» и две кнопки, позволяющие изменить цвет текста на зелёный или оранжевый.

## Описание приложения

Приложение представляет собой одностраничный веб-интерфейс.  
При нажатии на кнопки цвет текста изменяется без перезагрузки страницы.

---

## Используемые технологии

### Frontend

- **HTML5**  
  Используется для создания структуры страницы: текста и кнопок.

- **CSS3**  
  Применяется для оформления интерфейса (шрифты, отступы, выравнивание элементов).

- **JavaScript (Vanilla JS)**  
  Реализует логику изменения цвета текста при нажатии на кнопки.

**Почему выбран данный стек:**
- приложение не требует сложной логики;
- нет необходимости использовать фреймворки;
- код легко читается и быстро запускается в браузере;
- минимальные требования к окружению.

### Backend

Backend-часть в данном приложении не используется.

Это обусловлено тем, что:
- отсутствует работа с БД;
- нет авторизации и пользовательского ввода;
- нет необходимости выполнять серверные запросы;
- вся логика выполняется на стороне клиента.

Для запуска приложения достаточно:

Дополнительная настройка не требуется.

## Возможные уязвимости

Несмотря на простоту приложения, при публикации в открытом доступе могут возникнуть потенциальные риски:

- **XSS (Cross-Site Scripting)**  
  Возможна при добавлении пользовательского ввода без фильтрации данных.

- **Отсутствие Content Security Policy (CSP)**  
  Может позволить выполнение сторонних скриптов.

- **Отсутствие HTTPS**  
  При размещении на хостинге данные могут быть перехвачены.

- **Расширение логики без валидации**  
  При добавлении форм и запросов без проверок возможны уязвимости.

В текущей версии приложения риски минимальны.
